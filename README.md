# 悬浮剪贴板

一个 Windows 桌面悬浮常用语/快捷键工具。主面板和悬浮按钮默认置顶且不抢占当前输入焦点，点击按钮会把内容粘贴到当前正在编辑的位置。

当前版本使用极简浅色视觉：柔和灰背景、白色卡片、蓝色强调、大字号、无拖拽影子，适合作为常驻桌面工具。

## 运行方式

1. 安装 Python 3.8 或更高版本。
2. 可选安装快捷键录制库：

```powershell
pip install keyboard
```

3. 双击运行 `floating_clipboard.pyw`，或在命令行运行：

```powershell
python floating_clipboard.py
```

## 使用说明

- 点击主面板按钮：执行对应动作；文本动作会粘贴到上一个输入框并自动回车发送。
- 点击按钮会立即执行；如果没有记录到外部输入框，会提示先点击目标输入框，不会把内容发给工具窗口自己。
- 主面板快捷项按两列排列，使用卡片样式和轻量类型标识，适合大字号和常驻桌面使用。
- 标题栏 `_`：隐藏主窗口，并显示一个小圆形入口。
- 标题栏 `▾`：折叠/展开主面板，折叠后只保留标题栏。
- 标题栏最右侧 `◉`：缩成深色圆形剪贴板图标，短点击圆形图标可恢复，拖动或长按不会展开。
- 新增/编辑窗口使用明显的分段按钮选择类型，保存按钮固定在底部。
- 点击标题栏 `+`：新增常用语、快捷键或长按按键。
- 右键主面板按钮：编辑或删除。
- 拖拽主面板按钮到窗口外：生成桌面悬浮按钮。
- 点击悬浮按钮：执行对应动作。
- 拖动悬浮按钮：移动位置。
- 滚动悬浮按钮：调整宽度。
- 右键悬浮按钮：收回或设置小/中/大尺寸。

## 配置文件

配置保存在同目录的 `keyboard_config.json`：

```json
[
  {
    "name": "您好",
    "type": "text",
    "content": "您好，请问有什么可以帮您？"
  }
]
```

`type` 支持：

- `text`：写入剪贴板并模拟 `Ctrl+V`。
- `shortcut`：模拟组合键，例如 `ctrl+alt+a`。
- `hold`：点击一次按下，点击第二次释放。

## 注意

- 文本动作默认会在粘贴后自动按回车。需要只粘贴不发送时，可把 `floating_clipboard.py` 顶部的 `AUTO_ENTER_AFTER_TEXT` 改为 `False`。
- `keyboard` 库用于录制快捷键，在部分系统上需要管理员权限。即使未安装，也可以在编辑窗口手动输入快捷键文本。
- 执行日志保存在同目录的 `floating_clipboard.log`。如果没有按预期输入，可以打开这个文件查看完整错误。
